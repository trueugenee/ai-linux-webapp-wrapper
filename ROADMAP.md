# Roadmap

This project is intentionally small. The goal is not to become a full app
generator, but to stay a readable Linux-focused template for AI-assisted webapp
wrappers.

## Near Term

- Add more example configs for common webapps and dashboards.
- Improve first-run instructions for Linux desktop environments.
- Add optional app icon override support.
- Add a safer config validation error message when `app.config.json` is invalid.
- Document common login/OAuth host issues and how to add allowed hosts safely.

## Maybe Later

- Add a tiny config wizard.
- Add optional packaging examples for Arch Linux and Nix.
- Add screenshot or GIF preview to the README.
- Add optional per-profile app state directories.

## Not Planned

- A large GUI manager.
- A full replacement for Nativefier or Pake.
- Broad wildcard navigation rules by default.
- Renderer Node.js integration.
